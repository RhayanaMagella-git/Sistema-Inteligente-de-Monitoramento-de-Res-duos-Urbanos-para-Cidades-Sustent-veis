# Sistema-Inteligente-de-Monitoramento-de-Res-duos-Urbanos-para-Cidades-Sustent-veis
O sistema monitora o nível de resíduos em lixeiras urbanas com sensor ultrassônico HC-SR04. Ao atingir 90% da capacidade, LED e buzzer são acionados, enquanto os dados são enviados via MQTT para o broker HiveMQ na nuvem, permitindo monitoramento remoto em tempo real.
